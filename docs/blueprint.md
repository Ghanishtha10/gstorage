# **App Name**: Content Locker

## Core Features:

- Admin Authentication: Secure login mechanism for administrators to access content management features.
- Public Content View: Display a gallery or list of all uploaded files, accessible to anyone without login.
- File Upload (Admin Only): Admins can securely upload various file types (e.g., documents, images, videos) to the platform.
- File Deletion (Admin Only): Admins have the ability to remove uploaded files from the platform.
- AI Content Tagging Suggestion (Admin Tool): An AI tool analyzes uploaded file content (e.g., text from documents, metadata from images) and suggests relevant tags or categories to help admins organize content efficiently.
- Admin Dashboard: A dedicated interface for administrators to view, upload, delete, and manage all content and access suggested AI tags.
- Customizable Theme & Dark/Light Mode: Users can switch between dark and light modes and personalize the application's primary accent color.

## Style Guidelines:

- The app uses a modern dark theme to highlight content and provide a professional feel. The primary accent color is a muted cyan-blue, '#73B6CC', conveying reliability and technological polish. The background is a very dark, desaturated cyan, '#1F2D30', offering depth without being stark. A vibrant green, '#33CC84', is used for interactive elements and successful actions, providing clear visual cues.
- The app will support a toggle for Dark and Light modes. Additionally, users can choose from a palette of accent colors to personalize the interface.
- The font for all text, including headlines and body, is 'Inter' (sans-serif). Its clean, modern, and highly legible design ensures excellent readability across various content types and screen sizes.
- Utilize clear and universally recognizable iconography for key actions such as upload, delete, and view. Icons should maintain a consistent, minimalist aesthetic that complements the overall modern design.
- The admin interface features a clear two-column layout with a persistent sidebar for navigation and a main content area for file management. The public view presents content in a responsive grid or list format. Both layouts are designed to adapt seamlessly across different device screen sizes.
- Subtle and functional animations should enhance user interaction. This includes gentle hover effects on interactive elements and distinct loading indicators for file uploads and deletions, providing immediate visual feedback to the user.
